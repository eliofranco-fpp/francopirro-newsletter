# Franco, Pirro & Partners — Newsletter Archive

Archivio newsletter pubblicato su GitHub Pages.
Accessibile via: https://nl.francopirro.it

## Struttura
```
/                          ← Homepage archivio (index.html)
/2026-07-ai-act/           ← Newsletter luglio 2026 — AI Act
/CNAME                     ← Dominio personalizzato nl.francopirro.it
/.nojekyll                 ← Disabilita Jekyll processing
```

## Come aggiungere una nuova newsletter
1. Crea una cartella con naming: `YYYY-MM-argomento`
2. Copia il file HTML come `index.html` dentro quella cartella
3. Aggiungi la card corrispondente in `index.html` (sezione grid)
4. Commit e push → pubblicato in 1-2 minuti
